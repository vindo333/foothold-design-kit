# Foothold Design Kit

Public reference kit for external design/video tools (e.g. Claude Design).
Contains only marketing docs and the logo assets currently live in the
Foothold app UI — no application source code, credentials, or internal
data live here or ever will.

- `docs/MARKETING_DESIGN_SYSTEM.md` — visual design system (tokens, type, motion)
- `docs/POSITIONING.md` — brand positioning and audience
- `docs/PRODUCT_VIDEO_PLAN.md` — product video storyboard + shot list
- `docs/copywriting.md` — copywriting rules for all marketing copy
- `brand/` — current logo assets (dark/light wordmark, logomark)

This repo is a mirror maintained by hand from the private main repo. It is
not synced automatically; treat it as a snapshot for a specific design task.
